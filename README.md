# ClaudeEDA

An Exploratory Data Analysis (EDA) workspace developed with [Claude Code](https://claude.ai/code).

## About

This repository contains data analysis scripts, notebooks, and supporting files for exploratory data analysis projects. Work here is developed collaboratively using Claude Code as an AI assistant.

## Getting Started

### Prerequisites

- Python 3.x (or R, depending on the analysis)
- Jupyter Notebook / JupyterLab (for `.ipynb` files)

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/The-Ultimate-Bug-Maker/ClaudeEDA.git
   cd ClaudeEDA
   ```

2. Install dependencies (once a `requirements.txt` is added):
   ```bash
   pip install -r requirements.txt
   ```
